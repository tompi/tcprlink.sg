Adapt Hyper Bridge sample

This page allows different ways of launching the Adapt Hyper Bridge Course.

Simple Launch

Use the following to test just a single SCO:

  • index.html?void=1
  • index.html?svoid=1 (For testing with security cookie)

Hyper Player example

Use the following to test the player and manifest. Displaying the player can be usefull for debugging.

  • hyper/player/player.html?void=1&manifest=../../content/adapt-manifest-debug.json
  • hyper/player/player.html?svoid=1&manifest=../../content/adapt-manifest-debug.json (For testing with security cookie)

Hyper Player hidden menu example

Use the following to test the player and manifest. The menu will start up hidden.

  • hyper/player/player.html?void=1&manifest=../../content/adapt-manifest.json
  • hyper/player/player.html?svoid=1&manifest=../../content/adapt-manifest.json (For testing with security cookie)
